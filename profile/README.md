# Hi there 👋 — Welcome to Futurisource !

## 🙋 What is Futurisource ?

**Futurisource ** is an open source community where developers of **all levels** come together to design, build, and ship tools that make developers' lives easier.

We build:
- 🖥️ CLI utilities & terminal helpers
- 🔌 IDE plugins & extensions
- 🤖 AI-powered coding assistants
- 🔍 Code review & debugging tools
- 📄 Documentation generators
- ⚡ Dev productivity automations

> No idea too small. No contributor too new. If it helps a developer — it belongs here.

---

## 🌈 How can the community get involved?

### Step 1 — Join us on Discord
👉 **[discord.gg/futurisource](https://discord.gg/futurisource)**
Say hi in `#introductions` — we don't bite!

### Step 2 — Find something to work on
Browse our repos and look for issues tagged:
- `good first issue` — perfect for newcomers
- `help wanted` — we need your skills
- `idea` — vote on what to build next

### Step 3 — Propose a tool
Have an idea? Open a [New Tool Proposal](https://github.com/futurisource-/.github/issues/new) and the community votes on it!

### Step 4 — Read the contribution guide
📖 [CONTRIBUTING.md](./CONTRIBUTING.md) — takes 5 minutes, covers everything you need.

## 🏅 Contributor Levels

Everyone starts somewhere and grows over time:

```
🔰 Newcomer  →  🔧 Contributor  →  🏗 Builder  →  🔑 Core Member  →  🧠 Maintainer
```

See [CONTRIBUTING.md](./CONTRIBUTING.md) for how to level up and what perks come with each level.

## 🗓️ Community Rituals

| Event | When | What happens |
|---|---|---|
| 🛠️ **Tool Jam** | Monthly | 48-hour sprint to build something together |
| 🎯 **Showcase Friday** | Every Friday | Demo what you shipped in `#showcase` |
| 🗳️ **Roadmap Vote** | 1st of every month | Community votes on the next tool to build |
| 📣 **Async Standup** | Weekly | Post your update in `#standup` |
