# 🚀 Futurisource Labs

> **Building the tools that build the future — together.**

Futurisource Labs is an open source community where developers of all levels come together to design, build, and ship tools that make developers' lives easier. Whether you're writing your first pull request or your thousandth, there's a place for you here.

---

## 🌍 What We Build

We build **developer tools** — the kind that save time, reduce friction, and make coding more enjoyable. Think:

- CLI utilities & terminal helpers
- IDE plugins & extensions
- AI-powered coding assistants
- Code review & debugging tools
- Documentation generators
- Productivity scripts & automations

No idea is too small. If it helps a developer, it belongs here.

---

## 🧭 Our Values

| Value | What it means |
|---|---|
| **Open by default** | Everything we build is open source, always |
| **Beginner welcome** | No question is dumb. We were all beginners once |
| **Ship over perfect** | Working tools beat polished ideas |
| **Credit always** | We celebrate every contributor, big or small |
| **Community first** | Decisions are made together, not top-down |

## 🛠️ How to Get Involved

### 1. Join the Community
👉 [Join our Discord](https://discord.gg/futurisourcelabs) — introduce yourself in `#introductions`

### 2. Explore Projects
Browse our repositories and look for issues tagged:
- `good first issue` — perfect for newcomers
- `help wanted` — we need your skills
- `idea` — vote on what to build next

### 3. Propose a Tool
Have an idea? Open a [New Tool Proposal](https://github.com/futurisource-labs/.github/issues/new?template=tool-proposal.md) and the community votes on it.

### 4. Start Contributing
Read our [Contributing Guide](./CONTRIBUTING.md) — it takes 5 minutes and covers everything.

## 📁 Repository Structure

```
futurisource/
├── .github/          ← Org-level docs, templates, workflows
├── project-template/ ← Starter template for every new tool
├── [tool-name]/      ← Individual tool repositories
└── website/          ← Community website
```

## 🗓️ Community Rituals

| Event | When | What |
|---|---|---|
| **Tool Jam** | Monthly | 48-hr sprint to build something together |
| **Showcase Friday** | Every Friday | Demo what you shipped in `#showcase` |
| **Roadmap Vote** | 1st of month | Community votes on next tool to build |
| **Async Standup** | Weekly | Post your update in `#standup` |
